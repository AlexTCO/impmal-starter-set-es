# Imperium Maledictum - Starter Set (Castellano)

Traducción al castellano del módulo **Starter Set** de **Imperium Maledictum** para Foundry VTT.

## Requisitos

- [Imperium Maledictum](https://foundryvtt.com/packages/impmal) (sistema)
- [Imperium Maledictum - Starter Set](https://foundryvtt.com/packages/impmal-starter-set) (módulo oficial — de pago)
- [Babele](https://foundryvtt.com/packages/babele)
- [Imperium Maledictum - Traducción al Español (impmal-es)](https://github.com/GusRPG/impmal-es) (traducción base del sistema, recomendado)

## Instalación

Pega esta URL en Add-on Modules → Install Module → Manifest URL:

```
https://raw.githubusercontent.com/AlexTCO/impmal-starter-set-es/main/module.json
```

## Estructura

```
impmal-starter-set-es/
```
